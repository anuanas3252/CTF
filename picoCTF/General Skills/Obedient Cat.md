# Obedient Cat

## PROBLEM

> This file has a flag in plain sight (aka "in-the-clear"). Download flag.

## SOLUTION

1. Download the file
2. Open the terminal
3. Go to Downloads
4. Using cat command we read the file
5. Now you can see the Flag.
